Drop it into your app

Three files. No build step. Works wherever React + JSX run.

1. Files to copy
your-app/
├── color-picker.jsx        // component (~330 lines, no deps)
└── imperial-color-picker.css // styles (themable via CSS vars)
2. HTML — load it
<link rel="stylesheet" href="imperial-color-picker.css">
<script src="https://unpkg.com/react@18.3.1/umd/react.development.js"></script>
<script src="https://unpkg.com/react-dom@18.3.1/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
<script type="text/babel" src="color-picker.jsx"></script>
3. JSX — use it
function PaintButton() {
  const [color, setColor] = React.useState("#3b82f6");
  return (
    <ColorPicker
      value={color}
      onChange={(hex, alpha, [r, g, b]) => setColor(hex)}
    />
  );
}

API

Two props in. Three values out per change.

PropTypeDescription
valuestringInitial color as 6-digit hex (default #3b82f6)
onChange(hex, alpha, rgb) => voidFires on every drag/edit. hex is uppercase 6-digit. alpha is 0–1. rgb is [r, g, b] 0–255.

Theming

Override these CSS variables on .cp-root or any ancestor. Defaults follow prefers-color-scheme.

--cp-bg-hostInput background
--cp-fg-hostText color
--cp-border-hostInput borders
--cp-border-subtle-hostSliders & swatches
--cp-accent-hostFocus ring color
--cp-radius-hostCanvas corner radius
--cp-mono-hostMonospace font for inputs
