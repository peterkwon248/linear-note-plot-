# Plot 기능 카탈로그 — 네비게이션 & 검색 & 뷰

> 2026-03-30 대화 기반 전체 정리.

---

## 네비게이션

| 기능 | 설명 | 구현 비용 | 기존 인프라 |
|------|------|----------|-----------|
| ⌘Tab 퀵 스위처 | `Ctrl+Tab` → 최근 페이지 5개 오버레이 순환 | 매우 낮음 | navigationHistory |
| J/K 리스트 네비게이션 | J(아래)/K(위) 커서 이동, Enter로 열기. Linear 핵심 UX | 매우 낮음 | 키보드 핸들러 |
| Go To Anything (퍼지 파인더) | 파일명/헤딩/태그/위키 제목을 한 입력창에서 통합 검색 | 중간 | FlexSearch 통합 |
| 페이지 내 빠른 이동 (Ctrl+G) | 키보드로 TOC 팝업 → 타이핑 필터 → Enter 점프 | 낮음 | 플로팅 TOC 재활용 |
| 페이지 잠금 (Page Lock) | 완성된 문서 실수 수정 방지. Note 메타 `locked: boolean` | 매우 낮음 | 필드 1개 |
| 페이지 복제 (Duplicate) | 한 클릭 복사. `duplicatedFrom` 메타로 관계 추적 | 매우 낮음 | store 함수 |

## 검색 강화

| 기능 | 설명 | 구현 비용 | 기존 인프라 |
|------|------|----------|-----------|
| 검색 필터 (Scope) | "이 폴더 안에서만" / "위키에서만" / "최근 7일만" / "태그 안에서만" | 낮음 | FlexSearch 필터 |
| 정규식 검색 | 검색창에서 `/regex/` 패턴 지원 | 낮음 | 정규식 레이어 |
| 검색 결과 미리보기 | 매칭된 문맥 전후 50자 하이라이트 | 낮음 | FlexSearch snippet |
| 최근 검색 기록 | 최근 검색어 10개 저장. 검색창 클릭 시 표시 | 매우 낮음 | 로컬 스토리지 |
| 검색 결과 핀 | 특정 결과를 핀 → 같은 검색어로 재검색 시 최상단 | 낮음 | 핀 맵 |
| 역방향 검색 | "이 페이지가 어떤 검색어로 발견되는지" 표시 | 중간 | 역인덱스 |
| 시맨틱 유사 검색 (로컬) | TF-IDF 기반 "이 페이지와 비슷한 페이지" | 중간 | 단어 빈도 통계 |
| 글로벌 치환 (Find & Replace) | 전체 워크스페이스에서 텍스트 검색 + 일괄 치환 | 중간 | FlexSearch 확장 |
| 저널 뷰 (날짜 기반 검색) | "2024년 3월에 뭘 했지?" → 기간 타임라인 | 중간 | noteEvents |

## 뷰 시스템 확장

| 기능 | 설명 | 구현 비용 | 기존 인프라 |
|------|------|----------|-----------|
| 갤러리 뷰 | 커버 이미지 기반 카드 그리드. Pinterest 느낌 | 중간 | view-engine `viewMode: "gallery"` |
| 테이블 뷰 (정식) | 스프레드시트 스타일 메타데이터 한눈에 | 중간 | view-engine `viewMode: "table"` |
| 스마트 폴더 | SavedView를 사이드바에서 폴더와 동급 표시 | 낮음 | SavedView 이미 있음 |
| 그래프 필터 프리셋 | "노트만/위키만/이 태그만/최근 30일만" 프리셋 저장 | 낮음 | SavedView 패턴 |
| 그래프 노드 핀 | 특정 노드 위치 고정 | 매우 낮음 | ontologyPositions + pinned |
| 그래프 타임라인 재생 | createdAt 기준 노드가 하나씩 나타나는 애니메이션 | 중간 | SVG 애니메이션 |
| 그래프 클러스터 컬러링 | co-occurrence 기반 클러스터에 색상 부여 | 중간 | Louvain + 색상 매핑 |
| 관계 타입 시각화 차별화 | related-to=점선, extends=실선, contradicts=빨간선 | 낮음 | 엣지 스타일 분기 |
| 그래프 radial 레이아웃 (마인드맵 모드) | 선택 노드 중심 방사형 배치. d3.tree() | 낮음 | d3 레이아웃 전환 |
| 마인드맵 에디터 편집 (레벨 3) | 마인드맵 노드에서 직접 편집 → Heading 순서 동기화 | 높음 | 양방향 동기화 |
