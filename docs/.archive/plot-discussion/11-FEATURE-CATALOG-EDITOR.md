# Plot 기능 카탈로그 — 에디터 & 블록

> 2026-03-30 대화 기반 전체 정리. 총 ~200개 기능 아이디어 중 에디터/블록 영역.

---

## 에디터 UX

| 기능 | 설명 | 구현 비용 | 기존 인프라 |
|------|------|----------|-----------|
| Smart Paste | URL→링크카드, 이미지→압축저장, 표→Table, MD→리치텍스트, 코드→CodeBlock 자동 감지 | 낮음 | paste 핸들러 |
| 텍스트 확장 (Snippet) | `;today`→날짜, `;meet`→회의록 구조. Settings에서 단축어 등록 | 낮음 | 에디터 핸들러 |
| 클립보드 히스토리 | 최근 복사한 10개 기억, `/clipboard`로 삽입 | 낮음 | Clipboard API |
| 포커스 하이라이팅 (타이프라이터 모드) | 현재 편집 문장만 선명, 나머지 흐림. iA Writer 스타일 | 매우 낮음 | CSS opacity |
| 라이브 워드카운트 바 | 에디터 하단 `342 words · 1,847 chars · 2 min read` | 매우 낮음 | 상태바 |
| 멀티 커서 / 멀티 셀렉션 | `Ctrl+D`로 같은 단어 다중 선택 → 동시 편집 | 중간 | ProseMirror 플러그인 |
| Zen 디테일 모드 | 특정 섹션(Heading 단위)만 확대, 나머지 접힘 | 중간 | CSS + 상태 관리 |
| 에디터 탭 | 여러 페이지를 탭으로 열기. `openTabs: string[]` | 중간 | UI 레이어 |
| 읽기 시간 표시 | 페이지 상단 "약 3분 읽기". 단어수 ÷ 200 | 매우 낮음 | 텍스트 한 줄 |
| 가독성 점수 (한국어) | 문장 길이, 문단 길이, 수동태 비율. 규칙 기반 | 낮음 | 정규식 + 통계 |
| 글쓰기 목표 | "이 페이지 목표: 1,000단어" + 프로그레스 바 | 매우 낮음 | 메타 필드 1개 |
| 타이핑 사운드 | 기계식 키보드/타자기 소리. 끄기 가능 | 매우 낮음 | Web Audio API |
| 프레젠테이션 모드 | Heading 기준 슬라이드 분할 → 풀스크린 발표 | 중간 | CSS + 키보드 |
| Distraction-free 모드 | 사이드바 닫고 에디터 전체 화면 | 매우 낮음 | CSS |
| 스크롤 진행률 바 | 에디터 우측 얇은 프로그레스 바 | 매우 낮음 | CSS + scroll |
| 자동 저장 인디케이터 | "저장됨 ✓" / "저장 중..." / "변경사항 있음" | 매우 낮음 | UI 상태 |
| 포커스 모드 타이머 | `/focus 25min` → 뽀모도로. 사이드바 닫힘 + 타이머 | 매우 낮음 | setTimeout |
| 자동 괄호/따옴표 쌍 | `(` → `()` 자동 완성 | 매우 낮음 | 에디터 핸들러 |
| 드래그로 텍스트 이동 | 인라인 텍스트 선택 → 드래그로 위치 이동 | 매우 낮음 | 네이티브 브라우저 |
| 이모지 피커 | `/emoji` 또는 `:smile:` → 이모지 팔레트 | 낮음 | emoji-mart |
| 특수문자 삽입 | `/symbol` → ←→★☆♪©®™ 팔레트 | 낮음 | 정적 데이터 |
| 링크 미리보기 편집 | 링크 카드 제목/설명 수동 편집 | 낮음 | UI |
| 자동 목록 번호 매기기 | `a.`→`b.`, `i.`→`ii.` 알파벳/로마 숫자 목록 | 낮음 | TipTap 확장 |

## 콘텐츠 블록 추가

| 블록 | 설명 | 구현 비용 | 기존 인프라 |
|------|------|----------|-----------|
| Mermaid 다이어그램 | CodeBlock에 mermaid 언어 → 플로우차트/시퀀스/간트 자동 렌더 | 낮음 | CodeBlock 확장 |
| Excalidraw 드로잉 블록 | 에디터 인라인 미니 화이트보드. MIT 라이브러리 | 중간 | @excalidraw/excalidraw |
| 마인드맵 블록 | `/mindmap` → 현재 페이지 Heading 구조를 트리 시각화 | 중간 | TOC 데이터 + SVG |
| 인라인 쿼리 뷰 | `/query` → 노트 메타데이터 테이블/칸반/리스트 렌더 | 중간 | view-engine 재활용 |
| 토글 Heading | Heading + Toggle 결합. 클릭하면 해당 섹션 전체 접힘 | 낮음 | Heading attrs |
| Callout 변형 (Admonition) | Info/Warning/Error/Success/Tip/Quote 6가지 스타일 | 낮음 | Callout type 속성 |
| 프로그레스 바 블록 | `/progress 70%` → 시각적 프로그레스 바 | 매우 낮음 | SVG 한 줄 |
| 카운터 블록 | 클릭할 때마다 숫자 증가. 습관 트래킹/카운팅 | 낮음 | 인터랙티브 노드 |
| 평점 블록 | ⭐⭐⭐⭐☆ 5점 별점. 리스트에서 정렬 가능 | 낮음 | 커스텀 노드 |
| 타임라인 블록 | 날짜 + 이벤트 → 세로 타임라인 시각화 | 중간 | CSS + 데이터 |
| 비교 블록 (vs) | 두 항목 좌/우 비교. Columns 특수화 버전 | 낮음 | Columns 변형 |
| 투표/폴 블록 | `/poll` → 선택지 → 클릭 투표 → 결과 바 차트 | 낮음 | 커스텀 노드 |
| 접을 수 있는 인용 (Spoiler) | 기본 숨김, 클릭하면 공개. 스포일러/정답/힌트 | 낮음 | Toggle 변형 |
| 코드 실행 블록 | JS/Python 에디터 안에서 실행 + 결과 인라인 표시 | 높음 | Web Worker 샌드박스 |
| Kanban 인라인 블록 | 에디터 안 독립형 칸반. 블록 자체에 데이터 | 중간 | dnd-kit |
| 임베드 위젯 블록 | 시계, 카운트다운 타이머 미니 위젯 | 낮음 | 컴포넌트 |
| 이미지 주석/마크업 | 이미지 위에 화살표, 원, 텍스트 주석 | 중간 | Canvas API |

## 임베드 확장

| Google Maps | URL 감지 → iframe | 낮음 |
| Spotify 트랙 | URL 감지 → iframe | 낮음 |
