# 로컬퍼스트 노트앱의 모든 것 — 빠진 기능 목록

> 2026-03-30 대화 기반 정리
> 캐치프레이즈: "로컬퍼스트 노트앱으로서의 모든 것"

---

## Tier 1 — 없으면 "로컬퍼스트"라고 말하기 민망한 것들

### 오프라인/PWA

- 현재: Next.js 웹앱. IndexedDB에 데이터는 있지만, 서비스 워커 없으면 오프라인에서 앱 자체가 안 열림
- 필요: PWA manifest + 서비스 워커
- 구현 비용: 가벼움
- 의의: 로컬퍼스트의 가장 기본적인 약속. "인터넷 없이도 된다"

### Import/Export

- 현재 상태: TODO P1이지만 사실 P0급
- 최소 지원: Obsidian MD 폴더, Notion export, Evernote ENEX 가져오기 + Markdown/JSON 내보내기
- 의의: 데이터가 내 것이라는 신뢰가 로컬퍼스트의 핵심. 나갈 수 없는 데이터는 갇힌 것

### 로컬 백업/스냅샷

- 현재: Undo Manager가 세션 내 50단계. 세션 넘어가면 끝
- 필요: 하루 한 번 자동 IDB 스냅샷 → 로컬 파일 저장. "7일 전 상태로 복원" 기능
- 참고: 노션도 페이지 히스토리 제공, 옵시디언은 파일 자체가 버전 관리 가능

## Tier 2 — 경쟁앱 대비 확실한 차별화

### 데일리 노트

- Calendar 공간이 이미 있는데 데일리 노트가 없음
- 옵시디언/Logseq의 킬러 기능
- 구현: Calendar에서 날짜 클릭 → 해당 날짜 노트 자동 생성/열기. 템플릿 시스템 연결하면 "오늘의 노트" 템플릿 자동 적용
- Home 공간의 Today 섹션과 자연스럽게 연동

### Quick Capture

- 앱을 열지 않고 빠르게 메모하는 방법
- PWA: Share Target API로 모바일에서 공유 → Inbox 직행
- 데스크톱: 글로벌 단축키 → 미니 입력창
- 현재 마찰: 앱 열고 → Inbox 가고 → + 누르고 → 써야 함

### 웹 클리퍼

- TODO P1에 있음
- 브라우저 익스텐션으로 페이지/선택 텍스트 → Inbox
- `Note.source` 필드가 이미 `"webclip"` 타입을 지원하고 있으므로 파이프라인은 준비됨
- Home의 Inbox 요약 카드와 연동

### 인라인 쿼리 뷰

- `/query` 슬래시 커맨드로 에디터 안에서 노트 메타데이터 테이블 렌더
- 뷰 엔진(`runPipeline`)이 이미 있으므로 TipTap 커스텀 노드로 감싸기만
- 상세 설계: `03-INLINE-DB-DESIGN.md` 참조

## Tier 3 — "모든 것"을 완성하는 마지막 퍼즐

### PDF 뷰어/주석

- 로컬에 저장된 PDF를 앱 안에서 읽고 하이라이트
- 하이라이트를 노트로 추출
- 논문/책 읽는 사람한테는 핵심 기능
- 기술: pdf.js로 뷰어, 주석은 노트 + 관계로 연결

### 랜덤 노트 / Serendipity

- Discover 엔진이 이미 있음
- "랜덤으로 과거 노트 하나 보여주기" 기능
- SRS가 학습 기반 복습이라면 이건 우연한 재발견
- 구현: Home에서 "오늘의 재발견" 카드 또는 커맨드 팔레트에 "Surprise me" 추가
- 구현 비용: 10줄짜리 기능인데 체감 가치 큼

### 로컬 암호화

- 민감한 노트를 AES 암호화해서 IndexedDB에 저장
- 앱 열 때 비밀번호 입력 → 복호화
- 전체 암호화 or 노트별 잠금
- 로컬이니까 서버 키 관리 불필요 → 오히려 구현 단순

### Canvas / 프리폼 보드

- Obsidian Canvas, Apple Freeform 같은 것
- 노트 카드들을 공간 위에 자유롭게 배치
- Graph 뷰가 자동 배치라면 Canvas는 수동 배치
- React Flow (MIT) 사용 가능
- 스코프가 큼

## Tier 4 — 멀티디바이스 갈 때

### 로컬 P2P 싱크

- CRDTs (Yjs/Automerge) 기반으로 같은 와이파이 안에서 디바이스 간 동기화
- 서버 없이
- 또는 Git 기반 동기화
- "로컬퍼스트인데 여러 기기에서 쓸 수 있다"는 궁극의 약속
- 6개월+ 프로젝트

## Home 공간과의 연동

위 기능들 중 다수가 Home 대시보드와 자연스럽게 연결됨:

| Home 섹션 | 연동되는 기능 |
|----------|-------------|
| Today | 데일리 노트, Quick Capture |
| Inbox 요약 | 웹 클리퍼 결과, Quick Capture 결과 |
| Insights | 지식 건강 점수, 고립 노트, 관계 밀도 |
| Discover | 랜덤 노트, stub→article 승격 후보, Unlinked mentions |
| Recent | 최근 열어본 노트/위키 |
