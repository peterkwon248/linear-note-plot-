# Surface: 이슈 상세 / 에디터 (Issue Detail)

## Layer 1: 디자인 의사결정

### 구조: 콘텐츠 + 속성 사이드바
- 왼쪽: 제목 + 설명(에디터) + 활동 피드
- 오른쪽: 속성 사이드바 (`⌘/Ctrl I`로 토글)
- 속성 사이드바에서 모든 이슈 속성 수정 가능

### 에디터 특징
- 마크다운 기반, 리치 텍스트 편집
- `/` 슬래시 커맨드로 포매팅
- `@` 멘션 (사용자, 이슈, 프로젝트, 날짜, 문서)
- 링크 자동 임베드 (YouTube, Loom, Figma 등)
- 텍스트 하이라이트 → 플로팅 툴바

### 작성 후 3분 규칙
이슈 생성 후 3분 이내의 속성 변경은 "생성 과정"으로 간주, 활동 로그에 기록하지 않음. 사소한 수정이 히스토리를 오염시키지 않도록.

### 코멘트 전송
2026 리프레시에서 추가: `Enter` 또는 `⌘/Ctrl Enter` 중 선택 가능 (Preferences에서 설정).

## Plot 매핑
- TipTap 에디터에 `/` 슬래시, `@` 멘션 패턴 적용
- Detail Panel이 속성 사이드바 역할 (Relations, Backlinks, WIKI 등)
- 에디터 제목: text-[24px] font-semibold (현재 Plot 토큰과 일치)
