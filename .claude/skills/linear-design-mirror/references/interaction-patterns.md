## 4가지 액션 경로

Linear의 핵심 UX 패턴. 모든 액션을 4가지 방법으로 수행할 수 있다:

1. **직접 조작**: 클릭, 드래그, 속성 패널
2. **키보드 단축키**: 한두 키 조합 (예: `L`로 라벨, `A`로 담당자)
3. **컨텍스트 메뉴**: 우클릭으로 모든 가용 액션
4. **커맨드 팔레트**: ⌘K → 텍스트로 검색

예시 — 이슈에 라벨 "bug" 추가:
- 이슈 열기 → 속성 패널에서 라벨 클릭 → "bug" 선택
- 이슈 위에서 `L` → "bug" 선택
- 이슈 우클릭 → 라벨 → "bug"
- ⌘K → "label" 검색 → "bug" 선택

→ **Plot 적용 원칙**: 새 기능 추가 시 최소 2가지 접근 경로 제공. 키보드 단축키는 필수.

## 키보드 단축키 체계

### 패턴 기반 구조 (학습 가속)

**네비게이션: `G` + _** (Go to)
- `G` → `I`: 인박스
- `G` → `V`: 현재 사이클
- `G` → `B`: 백로그
- `G` → `A`: 활성 이슈
- `G` → `T`: 트리아지
- `G` → `X`: 아카이브

**메뉴 오픈: `O` + _** (Open)
- `O` → `F`: 즐겨찾기 목록
- `O` → `P`: 프로젝트 목록
- `O` → `C`: 사이클 목록
- `O` → `V`: 뷰 목록
- `O` → `T`: 팀 목록
- `O` → `I`: 최근 이슈

**이슈 액션 (단일 키)**
- `C`: 이슈 생성
- `V`: 전체 화면으로 이슈 생성
- `L`: 라벨
- `A`: 담당자
- `I`: 자신에게 할당
- `P`: 우선순위
- `S`: 상태
- `F`: 필터
- `E`: 예상치
- `X`: 선택/해제

**뷰 제어**
- `⌘/Ctrl B`: 보드/리스트 토글
- `⌘/Ctrl I`: 프로젝트 상세 사이드바 토글
- `⌘/Ctrl Shift I`: 인사이트 패널
- `⌘/Ctrl F`: 현재 뷰 내 검색
- `?`: 단축키 전체 목록

**이슈 이동 (보드)**
- `Option/Alt Shift ↑`: 컬럼 최상단으로
- `Option/Alt Shift ↓`: 컬럼 최하단으로

→ **Plot 적용**: 단축키를 패턴화. `G+_`=네비게이션, `O+_`=메뉴 오픈 패턴 차용.

## 커맨드 팔레트

### 동작 원리
- `⌘K` (Mac) / `Ctrl K` (Windows/Linux)로 열기
- **컨텍스트 의존**: 현재 뷰에 관련된 커맨드가 먼저 표시
- 퍼지 검색으로 모든 커맨드 접근
- 이슈, 프로젝트, 뷰 등 모든 엔티티 검색 가능

### 접두사 필터
커맨드 팔레트에서 특정 타입으로 포커스:
- `i` + space: 이슈 검색
- `p` + space: 프로젝트 검색
- `u` + space: 사용자 검색
- `t` + space: 팀 검색
- `l` + space: 라벨 검색
- `f` + space: 즐겨찾기 검색
- `d` + space: 문서 검색

### 디자인 세부사항
- 모달 형태, 메인 콘텐츠보다 약간 밝은 배경 (elevation 표현)
- 최근 사용 커맨드가 상단에 표시
- 각 커맨드 옆에 키보드 단축키 표시 (학습 도구 역할)
- edge lighting으로 모달의 z-axis 표현 (dark mode)

→ **Plot 적용**: 커맨드 팔레트는 Plot의 Search/Command Palette 서피스와 직접 대응.

## 리스트 네비게이션

### 이동과 선택
- `↑/↓` 또는 `J/K`: 하이라이트 이동
- `X`: 개별 선택 토글
- `Shift ↑/↓`: 범위 선택
- `Shift Click`: 범위 선택 (마우스)
- `⌘/Ctrl A`: 전체 선택

### 하이라이트 vs 선택의 구분
- **하이라이트**: 커서가 올라가 있는 상태. 단일 이슈에 대한 액션 가능.
- **선택**: 체크박스가 활성화된 상태. 다중 이슈에 대한 벌크 액션 가능.
- 하이라이트 상태에서 키보드 단축키 → 해당 이슈에 적용
- 선택 상태에서 키보드 단축키 → 선택된 모든 이슈에 적용

### 체크박스 노출
마우스를 이슈 왼쪽 끝(사이드바 경계 근처)으로 가져가면 체크박스 나타남.

→ **Plot 적용**: 노트 리스트에서 J/K 네비게이션, X 선택 패턴 적용 가능.

## Peek 미리보기

macOS Quicklook에서 영감. "Linear UI의 반쯤 비밀인 기능."

- `Space` 탭: Peek 토글 (on/off)
- `Space` 홀드: 누르고 있는 동안만 Peek 활성
- Peek 활성 상태에서 `J/K`로 이동하면 미리보기가 자동 갱신
- 커맨드 팔레트(⌘K)에서도 프로젝트/이슈에 Quicklook 가능

표시 정보: 설명, 담당자, 상태, 우선순위, 사이클, 라벨, 예상치, 생성/수정일

→ **Plot 적용**: 노트 리스트에서 Space로 미리보기 토글 — 에디터를 열지 않고도 내용 확인.

## 컨텍스트 메뉴

### safe area 구현
이슈를 우클릭하면 컨텍스트 메뉴 표시. 서브메뉴가 있을 때 마우스가 서브메뉴 방향으로 이동하는 중 메뉴가 닫히지 않도록 삼각형 safe area를 그린다.

구현: React 컴포넌트가 서브메뉴를 ref로 받아 크기/위치를 측정, 마우스 커서와 서브메뉴 사이에 삼각형을 그려 안전 영역 생성.

### 가용 액션
컨텍스트 메뉴에서 이슈에 대한 거의 모든 액션 가능:
- 상태/우선순위 변경
- 담당자 변경
- 예상치 설정
- 블로킹/블로커 표시
- 중복 표시
- 사이클/프로젝트에 추가
- Git 브랜치명 복사
- 이슈 URL 복사

→ **Plot 적용**: 노트/폴더에 대한 컨텍스트 메뉴 구현 시 safe area 패턴 필수.

## 인박스/트리아지 패턴

### 인박스
- 사이드바 최상단 위치
- `G` → `I`로 바로 이동
- 스플릿 뷰: 왼쪽에 알림 리스트, 오른쪽에 이슈 상세
- `J/K`로 알림 간 이동
- `Backspace`: 알림 삭제
- `Shift Backspace`: 전체 삭제
- `U`: 읽음/안읽음 토글
- `H`: 스누즈
- `⌘/Ctrl F`: 인박스 내 검색 (제목, ID, 알림 유형, 담당자)

### 트리아지
- 팀별 특수 인박스
- 인테그레이션 또는 외부 팀이 만든 이슈가 자동으로 도착
- 수락 전 검토/업데이트/우선순위 설정 기회
- Triage Intelligence: AI가 담당자, 라벨, 관련 이슈를 제안
- 스누즈: 지정 시간 또는 새 활동 시 재등장

→ **Plot 적용**: 인박스의 스플릿 뷰 패턴, 스누즈 개념은 SRS 리마인더 시스템과 유사.

## 검색 패턴

### 글로벌 검색
- 사이드바 돋보기 아이콘 또는 `/`로 열기
- 전체 워크스페이스의 이슈, 문서를 제목/설명/코멘트에서 검색
- 최근 검색 + 최근 이슈 표시

### 뷰 내 검색
- `⌘/Ctrl F`: 현재 뷰의 이슈를 제목으로 필터 (임시 필터처럼 동작)
- 타이핑하면 매칭되지 않는 이슈가 사라짐
- `Esc`로 검색 해제하면 전체 이슈 복원

### 검색 vs 커맨드 팔레트
- 검색: 콘텐츠 찾기에 최적화
- ⌘K: 액션 수행에 최적화
- 둘 다 퍼지 매칭 지원

→ **Plot 적용**: 글로벌 검색(FlexSearch)은 이미 있음. 뷰 내 실시간 필터링 패턴 추가 고려.

## Display Options 패턴

모든 뷰에 존재. 뷰 우상단에 위치.

### 기능
- 레이아웃 전환 (리스트 ↔ 보드)
- 그룹핑 (상태, 담당자, 프로젝트, 우선순위, 사이클, Focus)
- 정렬 순서
- 표시할 속성 선택
- 개인 설정 또는 워크스페이스 기본값으로 저장

### "Set as default" vs 개인 설정
- 커스터마이즈만 하면: 개인 설정 (다른 뷰로 갔다 와도 유지)
- "Set as default" 누르면: 워크스페이스 전체에 적용
- "Reset to default": 기본 설정으로 복원

→ **Plot 적용**: 노트 리스트의 표시 옵션 (정렬, 그룹핑)에 같은 패턴 적용 가능.

## 에디터 패턴

### 포매팅
- 키보드 단축키 또는 텍스트 하이라이트 시 툴바로 스타일 적용
- `/` 입력으로 포매팅 옵션 메뉴 (슬래시 커맨드)
- `@` 입력으로 멘션 (사용자, 이슈, 프로젝트, 날짜, 문서)
- `:` 입력으로 이모지 검색

### 임베드
- YouTube, Descript, Loom 등의 링크를 자동 감지하여 임베드
- Figma 링크 → 파일 미리보기 임베드 (인테그레이션 설정 시)
- 임베드 대신 링크로 유지하고 싶으면: "Keep as link" 또는 `Esc`

### 이슈 생성
- `C`: 이슈 생성 모달
- `V`: 전체 화면 이슈 생성
- `Option/Alt C`: 템플릿에서 생성
- 텍스트 하이라이트 후 `C`: 하이라이트된 텍스트가 제목으로 프리필
- 작성 후 3분 이내 속성 변경은 "생성 과정"으로 간주, 활동 로그에 변경으로 기록 안 함

→ **Plot 적용**: TipTap 에디터에 `/` 슬래시 커맨드, `@` 멘션 패턴 적용.
